Formulario

1. Escribir un programa en JavaScript que pregunte al usuario por su nombre y le salude, mediante una ventana emergente, con el mensaje “Hola nombreUsuario”.

2. Escribe un programa en JavaScript que pida al usuario que introduzca una cadena de texto. Mediante una ventana emergente, el programa mostrará cuántas letras ‘a’ hay en la cadena de texto introducida.

4. Escribe un programa en JavaScript que pregunte al usuario qué color de fondo prefiere. El usuario responderá ‘R’ para rojo, ‘V‘ para verde o ‘A’ para azul. Una vez respondido, el fondo del documento cambiará al color escogido.

5. Escribe un programa en JavaScript que, a partir de tres botones como los de la imagen, cambie el color de fondo según el botón pulsado. Deberéis implementar el código mediante una función a la que se le pase el color escogido como parámetro de entrada.

6. Escribe un programa en JavaScript que contenga un elemento div . Al pasar el ratón por encima de dicho elemento, deberá cambiar su color de fondo (solamente el del div) a cualquier color que queráis.

8.Diseña un programa en JavaScript que permita al usuario introducir tres valores numéricos en un pequeño formulario. Al enviar el formulario, el programa informará, mediante ventana emergente, de cuál de los tres números introducidos es el mayor. Nota: el programa nunca seleccionará el número mayor si alguno de los campos está en blanco o el valor que contiene no es numérico
